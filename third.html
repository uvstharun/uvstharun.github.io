Credit Card Default Prediction: Unlocking Credit Risk with Machine Learning

Managing credit risk is a critical aspect of the financial industry, and predicting credit card default is one of the most challenging tasks in this domain. In this project, I explore various machine learning techniques to predict the likelihood of credit card default among clients, providing valuable insights into risk management. By comparing the performance of four different models, I aim to provide data-driven recommendations to improve creditworthiness assessments and help financial institutions make informed lending decisions.

Project Overview

The objective of this project is to predict whether a client will default on their credit card payment for the next month. I implemented four machine learning models — Logistic Regression, K-Nearest Neighbors (KNN), Classification and Regression Trees (CART), and Random Forest — and analyzed each model’s effectiveness in terms of accuracy, sensitivity, and specificity. This comparison highlights which model performs best for credit risk management.

Models & Techniques

Logistic Regression

Used feature selection techniques such as Forward Selection, Backward Elimination, and Stepwise Selection with StepAIC.

Key Insight: As the threshold for prediction decreases, sensitivity increases while specificity decreases. All feature selection methods returned similar accuracy results.

K-Nearest Neighbors (KNN)

Optimized the K-value using cross-validation to find the best performance.

Key Insight: KNN performed best with K = 161, balancing accuracy, sensitivity, and specificity.

Classification and Regression Trees (CART)

Used complexity parameter pruning to prevent overfitting.

Key Insight: The pruned tree generalized better, although the full tree initially had higher accuracy.

Random Forest

Built an ensemble of decision trees for more robust predictions.

Key Insight: Random Forest achieved the highest accuracy of 99% on the training set and 84% on the test set, making it the most accurate model overall.

Model Performance Summary

Model Accuracy (Test Set) Sensitivity Specificity
Logistic Regression 75.66% 77.8% 69.7%
K-Nearest Neighbors (KNN) 83.4% 51.9% 75.7%
CART (Pruned) 70.61% 33.7% 90.2%
Random Forest 84% 76.4% 83.3%

Key Insights & Recommendations

Logistic Regression: Offers a balance between interpretability and performance, making it ideal for risk management.

Random Forest: Provides the highest accuracy, making it suitable for large-scale credit default prediction.

KNN & CART Require further tuning but can still provide valuable insights for smaller datasets or specific tasks.

Futuer Work

In the future, the project can be enhanced by focusing on hyperparameter optimization, particularly for models like Random Forest and K-Nearest Neighbors (KNN), to fine-tune their performance and achieve higher accuracy. Additionally, using ensemble methods, such as combining multiple models, could lead to more robust predictions by leveraging the strengths of different algorithms. Finally, expanding the feature set by incorporating more financial or behavioral data can improve the model’s predictive power, leading to more accurate and reliable outcomes for credit risk assessment.
